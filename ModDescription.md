Harmony is the shared patch library used by many RimWorld mods.

This mod provides Harmony to the game so other mods can depend on one common version instead of each shipping their own copy.

For players, it is usually simple: install Harmony, keep it high in the mod list, and let other mods use it.

For mod authors, this keeps patching cleaner and reduces duplicate `0Harmony.dll` copies in mod folders.
